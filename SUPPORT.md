# Support

For bugs, feature requests, and usage questions, open a GitHub issue in the repository
that hosts `claude-frontend`.

Include:

- command used, including flags
- frontend framework and package manager
- relevant file paths or target URL
- whether Playwright/browser validation was available
- screenshots, console errors, or build output if the problem is visual or runtime-related

For security issues, follow [SECURITY.md](SECURITY.md).
